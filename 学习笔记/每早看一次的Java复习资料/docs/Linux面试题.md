## 检索文件内容 grep 
grep "被查找的字符串" 文件名
`grep -n 2019-10-24 00:01:11' *.log`
可以.log文件中，查到时间内的所有信息

参数：
-o 或 --only-matching : 只显示匹配PATTERN 部分。
-v 或 --revert-match : 显示不包含匹配到的文本的所有行。  （过滤掉自己当前操作的信息，就是-v后面的内容）
### 管道操作符    |
可将指令连接起来,前一个指令的输出作为后一个指令的输入
